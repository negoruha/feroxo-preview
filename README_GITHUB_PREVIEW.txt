FEROXO — GITHUB PREVIEW

This package contains the full static site preview: Home, Contact, Products, product details, Resources, category and article pages, as well as the Make Request modal.

GitHub Pages preview checks:
- desktop and mobile layout
- navigation and language menu
- cards, search, sliders and modal opening
- product prefill in the request modal
- document placeholder links and video links

The PHP form handler is included for FORPSI. GitHub Pages is only for visual and interaction testing; it does not run PHP mail sending.

The included PowerShell script creates a new public GitHub repository, turns on Pages from main / root, and opens the preview URL.
